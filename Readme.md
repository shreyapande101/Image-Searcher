#  Image Searcher Platform

This is a simple **Image Searcher Platform** project made using **HTML, CSS, and JavaScript**.  
It allows users to search for images by typing keywords and displays related images on the screen.

---

 Features

- Search images using keywords
- Clean and simple user interface
- Beginner-friendly project

---
 Technologies Used

- **HTML** – for structure  
- **CSS** – for styling and layout  
- **JavaScript** – for fetching api and displaying images  

---

 How to Use

1. Type a keyword in the search box  
2. Click on the **Search** button  
3. Related images will be displayed on the screen
